# Security Policy

## Reporting a Vulnerability

Please do not open a public GitHub issue for security vulnerabilities.

**Preferred:** use [GitHub private vulnerability reporting](https://github.com/memoriqme/memoriq-extension/security/advisories/new) on this repository (Repository → Security → Report a vulnerability).

**Email:** if you cannot use GitHub, use the contact details published on [memoriq.me](https://memoriq.me).

Helpful details include:

- affected version or commit
- affected deployment type: hosted, self-hosted, browser extension, or local development
- steps to reproduce
- expected impact
- whether any user data, extension token, or encrypted vault material may be affected

We will acknowledge valid reports as soon as practical and coordinate disclosure when a fix is available.
